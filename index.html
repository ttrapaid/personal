AHRQ SDOH CHALLENGE

DATA VISUALIZATION CHALLENGE
MARCH 2019 - PRESENT

The Agency for Healthcare Research and Quality (AHRQ) Visualization Resources of Community-Level Social Determinants of Health Challenge asked participants to create tools that support visualizing data clusters to enhance the research and analysis of community-level health services.

Challenge participants were tasked with developing visualization tools that can augment the insights drawn from the analysis of medical expenditure and health care utilization data at the community level.

I architected and implemented a solution to the challenge using an open source system called Superset, which is managed by Apache. I created data ingestion endpoints to import publicly available data from various sources and created dashboards using Superset to visualize this data. I deployed this solution using containers onto the AWS cloud.

At the time of writing this, I am a Phase 1 winner of this challenge.

MONITORING THE ENVIRONMENT

UMD ENVIRONMENTAL MONITORING LAB
OCTOBER 2018 - PRESENT

This project aims to monitor environmental conditions using a low cost smart sensor network. Carbon dioxide and other particulate matter sensors are connected to Raspberry Pis that are placed across the UMD campus, throughout the city of College Park, and the surrounding Greenbelt area. Python scripts are used to measure and send back the recorded data to our central repository.
